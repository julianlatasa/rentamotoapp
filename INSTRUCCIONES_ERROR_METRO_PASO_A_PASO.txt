INFORMACIÓN TÉCNICA (Por si necesitas debuggear)
═════════════════════════════════════════════════

El error ocurre porque:
- Metro no puede leer el archivo metro-runtime/require.js
- Puede ser caché corrupta, permisos, o sincronización

Solutions:
- npm cache clean --force → Limpia caché de npm
- npm start --reset-cache → Reconstruye Metro desde cero
- rmdir node_modules → Reinstala todas las dependencias

Es común después de:
- Cambios grandes en dependencias
- Interrupciones durante npm install
- Actualizaciones de React Native
- Cambios en metro.config.js
