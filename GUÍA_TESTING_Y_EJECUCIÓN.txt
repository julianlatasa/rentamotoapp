ERRORES COMUNES Y SOLUCIONES
─────────────────────────────

Error: "Cannot find module 'react-native-paper'"
→ Ejecuta: npm install react-native-paper

Error: "BottomNavigation is not exported"
→ Verifica que imports en App.tsx sean correctos
→ Necesitas: import { BottomNavigation } from 'react-native-paper';

Error: "Module not found: OffersScreen"
→ Verifica ruta correcta: src/screens/OffersScreen.js
→ Verifica nombre exacto del archivo

Error: "Can't find variable UserContext"
→ Verifica que AuthContext esté correctamente importado
→ Verifica que el provider enuelva la app

Iconos no aparecen:
→ Verifica que react-native-vector-icons esté instalado
→ En Android: Puede necesitar rebuild
→ En iOS: Puede necesitar pod install

Scroll no funciona en Chat:
→ Verifica que el componente esté dentro de <KeyboardAvoidingView>
→ Verifica que <FlatList> tenga ref y contentContainerStyle
